Getting Started Angular.js

Coleção que reúne pequenos projetos demonstrativos para reforçar meu conhecimento em Angular. Cada projeto aplicará conceitos fundamentais de Angular, como Components, Inputs, Outputs, data binding, serviços, roteamento e diretivas. Ao final, o site servirá como uma referência prática e visual de como utilizar as principais funcionalidades do Angular em projetos reais.
